# JamJar

JamJar dynamically creates SQLite-backed, in memory, ActiveRecord models to help you test your ActiveRecord extensions.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "jamjar"
```

And then execute:

    $ bundle
